    Activity4.2Graphical Derivatives

    Much information about a function's first derivative can be gleaned simply by looking at a graph of the function. In fact, a person with good visual skills can “see” the graph of the derivative while looking at the graph of the function. This activity focuses on helping you develop that skill.

    Exercises4.2.1Exercises

    A parabolic function is shown in Figure 4.2.1. The next several questions are in reference to that function.

    Exercise 1

    Several values of the function \(\fd{g}\) are given in Table 4.2.3. For each given value draw a nice long line segment at the corresponding point on \(g\) whose slope is equal to the value of \(g\). If we think of these line segments as actual lines, what do we call the lines?

    Exercise 2

    What is the value of \(\fd{g}\) at \(1\)? How do you know? Go ahead and enter that value into Table 4.2.3.

    Exercise 3

    The function \(g\) is symmetric across the line \(x=1\); that is, if we move equal distance to the left and right from this line the corresponding \(y\)-coordinates on \(g\) are always equal. Notice that the slopes of the tangent lines are “equal but opposite” at points that are equally removed from the axis of symmetry; this is reflected in the values of \(\fe{\fd{g}}{1}\) and \(\fe{\fd{g}}{3}\). Use the idea of “equal but opposite slope equidistance from the axis of symmetry” to complete Table 4.2.3.

    Exercise 4

    Plot the points from Table 4.2.3 onto Figure 4.2.2 and connect the dots. Determine the formula for the resultant linear function.

    Exercise 5

    The formula for \(g\) is \(-0.5x^2+x+5.5\). Use Definition 3.3.1 to determine the formula for \(\fe{\fd{g}}{x}\).

    Exercise 6

    The line you drew onto Figure 4.2.2 is not a tangent line to \(g\). Just what exactly is this line?

    A function \(f\) is shown in Figure 4.2.10 and the corresponding first derivative function \(\fd{f}\) is shown in Figure 4.2.11. Answer each of the following questions referencing these two functions.

    Exercise 7

    Draw the tangent line to \(f\) at the three points indicated in Figure 4.2.10 after first using the graph of \(\fd{f}\) to determine the exact slope of the respective tangent lines.

    Exercise 8

    Write a sentence relating the slope of the tangent line to \(f\) with the corresponding \(y\)-coordinate on \(\fd{f}\).

    Exercise 9

    Copy each of the following sentences onto your paper and supply the words or phrases that correctly complete each sentence.

    • Over the interval where \(\fd{f}\) is negative, \(f\) is …
    • Over the interval where \(\fd{f}\) is positive, \(f\) is …
    • Over the interval where \(\fd{f}\) is increasing, \(f\) is …
    • Over the interval where \(\fd{f}\) is decreasing, \(f\) is …
    Exercise 10

    In each of Figures 4.2.16 and 4.2.17 a function (the solid curve) is given; both of these functions are symmetric about the \(y\)-axis. The first derivative of each function (the dash-dotted curves) have been drawn over the interval \(\ointerval{0}{7}\). Use the given portion of the first derivative together with the symmetry of the function to help you draw each first derivative over the interval \(\ointerval{-7}{0}\).

    A graph of the function \(f\) given by \(\fe{f}{x}=\frac{1}{x}\) is show in Figure 4.2.18.

    Exercise 11

    Except at \(0\), there is something that is always true about the value of \(\fd{f}\); what is the common trait?

    Exercise 12

    Use Definition 3.3.1 to find the formula for \(\fe{\fd{f}}{x}\).

    Exercise 13

    Does the formula for \(\fe{\fd{f}}{x}\) support your answer to Exercise 4.2.1.12?

    Exercise 14

    Use the formula for \(\fe{\fd{f}}{x}\) to determine the horizontal and vertical asymptotes to the graph of \(y=\fe{\fd{f}}{x}\).

    Exercise 15

    Keeping it simple, draw onto Figure 4.2.19 a curve with the asymptotes found in Exercise 4.2.1.14 and the property determined in Exercise 4.2.1.11. Does the curve you drew have the properties you would expect to see in the first derivative of \(f\)? For example, \(f\) is concave down over \(\ointerval{-\infty}{0}\), and concave up over \(\ointerval{0}{\infty}\); what are the corresponding differences in the behavior of \(\fd{f}\) over those two intervals?

    Exercise 16

    A graph of the function \(g\) is shown in Figure 4.2.26. The absolute minimum value ever obtained by \(\fd{g}\) is \(3\). With that in mind, draw \(\fd{g}\) onto Figure 4.2.27. Make sure that you draw and label any and all necessary asymptotes. Make sure that your graph of \(\fd{g}\) adequately reflects the symmetry in the graph of \(g\).

    A function, \(w\), is shown in Figure 4.2.28. A larger version of Figure 4.2.28 is available in <<Appendix B (page B3)>>. Answer each of the following questions in reference to this function.

    Exercise 17

    An inflection point is a point where the function is continuous and the concavity of the function changes. The inflection points on \(w\) occur at \(2\), \(3.25\), and \(6\). With that in mind, over each interval stated in Table 4.2.29 exactly two of the properties from its caption apply to \(\fd{w}\). Complete Table 4.2.29 with the appropriate pairs of properties.

    Exercise 18

    In Table 4.2.32, three possible values are given for \(\fd{w}\) at several values of \(x\). In each case, one of the values is correct. Use tangent lines to \(w\) to determine each of the correct values. (This is where you probably want to use the graph on <<page B3>>.)

    Exercise 19

    The value of \(\fd{w}\) is the same at \(2\), \(4\), and \(7\). What is this common value?

    Exercise 20

    Put it all together and draw \(\fd{w}\) onto Figure 4.2.33.
