Definition3.3.1The First Derivative Function

If \(f\) is a function of \(x\), then we define the first derivative function, \(\fd{f}\), as \[\fe{\fd{f}}{x}=\lim_{h\to0}\frac{\fe{f}{x+h}-\fe{f}{x}}{h}\text{.}\] The symbols \(\fd{f}\) are read alound as “\(f\) prime of \(x\)” or “\(f\) prime at \(x\)”.
