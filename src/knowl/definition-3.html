Definition2.10.1One-sided Continuity

The function \(f\) is continuous from the left at \(a\) if and only if

\[\lim\limits_{x\to a^{-}}\fe{f}{x}=\fe{f}{a}\text{.}\]

The function \(f\) is continuous from the right at \(a\) if and only if

\[\lim\limits_{x\to a^{+}}\fe{f}{x}=\fe{f}{a}\text{.}\]
