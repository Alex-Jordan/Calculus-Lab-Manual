Solution 1.4.1.0.2.1

The rise between the two points is \(9\) and the run is \(3\), so the slope between the two points is given by \(\frac{9}{3}=3\).

Using the difference quotient, if we let \(x=-1\) and \(h=3\) we get

\begin{align*} 4-2x-h&=4-2(-1)-3\\ &=3\quad\checkmark \end{align*}
