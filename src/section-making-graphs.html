    Activity9.7Making Graphs

    Let's put it all together and produce some graphs.

    Exercises9.7.1Exercises

    Consider the function \(\fe{f}{x}=\frac{8x^2-8}{(2x-4)^2}\).

    Exercise 1

    Evaluate each of the following limits: \(\lim\limits_{x\to\infty}\fe{f}{x}\), \(\lim\limits_{x\to-\infty}\fe{f}{x}\), \(\lim\limits_{x\to2^{-}}\fe{f}{x}\), and \(\lim\limits_{x\to2^{+}}\fe{f}{x}\).

    Exercise 2

    What are the horizontal and vertical asymptotes for a graph of \(f\)?

    Exercise 3

    What are the horizontal and vertical intercepts for a graph of \(f\)?

    Exercise 4

    Use the formulas \(\fe{\fd{f}}{x}=\frac{4(1-2x)}{(x-2)^3}\) and \(\fe{\sd{f}}{x}=\frac{4(4x+1)}{(x-2)^4}\) to help you accomplish each of the following.

    • State the critical numbers of \(f\).
    • Create well-documented increasing/decreasing and concavity tables for \(f\).
    • State the local minimum, local maximum, and inflection points on \(f\). Make sure that you explicitly address all three types of points whether they exist or not.
    Exercise 5

    Graph \(y=\fe{f}{x}\) onto Figure 9.7.6. Make sure that you choose a scale that allows you to clearly illustrate each of the features found in Exercises 9.7.1.1–9.7.1.4. Make sure that all axes and asymptotes are well labeled and also write the coordinates of each local extreme point and inflection point next to the point on the graph.

    Exercise 6

    Check your graph using a graphing calculator.

    Consider the function \(\fe{g}{t}=\frac{1}{(e^t+4)^2}\).

    Exercise 7

    Evaluate each of the following limits: \(\lim\limits_{t\to\infty}\fe{g}{t}\) and \(\lim\limits_{t\to-\infty}\fe{g}{t}\).

    Exercise 8

    What are the horizontal and vertical asymptotes for a graph of \(g\)?

    Exercise 9

    What are the horizontal and vertical intercepts for a graph of \(g\)?

    Exercise 10

    Use the formulas \(\fe{\fd{g}}{t}=\frac{-2e^t}{(e^t+4)^3}\) and \(\fe{\sd{g}}{t}=\frac{4e^t(e^t-2)}{(e^t+4)^4}\) to help you accomplish each of the following.

    • State the critical numbers of \(g\).
    • Create well-documented increasing/decreasing and concavity tables for \(g\).
    • State the local minimum, local maximum, and inflection points on \(g\). Make sure that you explicitly address all three types of points whether they exist or not.
    Exercise 11

    Graph \(y=\fe{g}{t}\) onto Figure 9.7.13. Make sure that you choose a scale that allows you to clearly illustrate each of the features found in Exercises 9.7.1.7–9.7.1.10. Make sure that all axes and asymptotes are well labeled and also write the coordinates of each local extreme point and inflection point next to the point on the graph.

    Exercise 12

    Check your graph using a graphing calculator.
